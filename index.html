Sitio web Auto.cl

Desarrollo Front end para cliente Auto.cl. Se utilizó Angular 14 y Angular Material como framework UI.

El requerimiento principal fue crear flujos para que los clientes pudieran elegir un auto que se acomode a sus preferencias. Esto se logró a través de formularios compuestos de distintos pasos que resultaban en las coincidencias encontradas.

Visitar www.auto.cl

Framework Kaufmann

Se crea un sistema de componentes UI para reutilizar en sus distintos proyectos Angular y Liferay.

El proyecto se construyó con Angular 14 y Ant design. Se creó una librería NPM compartida a través de Artefactos de Azure. Para las instrucciones de uso y detalles de cada componente se implementó la documentación con Storybook.

Visitar www.kaufmann.cl
